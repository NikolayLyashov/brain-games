# frontend-project-lvl1

## Setup

    $ make install
    $ make build

## Games

    * Calculation
        * $ brain-calc
    * Check parity
        * $ brain-even
    * Greatest common divisor
        * $ brain-gcd
    * Arithmetic progression   
        * $ brain-progression
    * given number is prime?
        * $ brain-prime
